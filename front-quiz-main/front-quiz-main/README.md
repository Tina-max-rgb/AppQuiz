# Q+ Quiz — Maquette fidèle (Vue 3 + Vite + Tailwind)

Ce projet reproduit l'ensemble des écrans de la maquette fournie : connexion, dashboard, création de quiz, gestion des questions/réponses, administration, participation.

## Démarrage

```bash
npm install
npm run dev
```

## Notes
- Projet front-end avec données simulées (mock).
- Couleurs principales : bleu (primary), vert (success), rouge (danger).
- Polices : Inter via Google Fonts.
